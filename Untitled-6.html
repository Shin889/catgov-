FOR JOB SEEKERS

How to Register as Applicant

  • Go to https://pesojobs.ph/ and select REGISTER NOW
  • Select APPLICANT and fill out the registration form
  • Accept terms and create your account
  • After login, select CURRICULUM VITAE to build or update your CV
  • Complete BASIC PROFILE, EDUCATIONS, WORK EXPERIENCE, and SKILLS sections
Register Now

How to Build Your CV

  • Go to https://pesojobs.ph/ and login
  • Select CURRICULUM VITAE
  • Update BASIC PROFILE, EDUCATIONS, WORK EXPERIENCE, and SKILLS
  • Save changes after each section
Build CV

How to Search and Apply for Jobs

  • Using your web browser, enter the URL https://pesojobs.ph/
  • Select JOB SEARCH to search for job vacancies. Select SEARCH to manually search the name of the employer / job position
  • Select VIEW to view the details / job post of the employer
  • Select APPLY to apply for the selected job vacancy
Search Jobs

FOR EMPLOYERS

How to Register as Employer

  • Using your web browser, enter the URL https://pesojobs.ph/
  • You will be redirected to the login / registration page of the system
  • Select REGISTER NOW to create an account
  • Select EMPLOYER. Enter all the information required in the registration form
  • Select ACCEPT OUR TERMS AND CONDITIONS. Select CREATE ACCOUNT to proceed
  • You will be redirected to the home page of the system, after successful login
Register as Employer

How to Update Company Details

  • After successfully registering, provide or update your company details subject for approval of the PESO to be able to post job vacancies
  • Select your PROFILE PICTURE / ICON and select COMPANY DETAILS
  • Enter all the required information to update your company details & information
  • Select if a Licensed Overseas Recruitment Agency (LRA) or Private Employment Agency (PEA), then enter your business address / location, contact information and upload your company logo. Select SUBMIT button to finalize
Update Details

How to Upload Legal Documents

  • Select LEGAL DOCUMENTS to attach your company's legal documents subject for review and approval of PESO
  • Select CHOOSE FILE to upload a document and enter the DATE ISSUED and DATE EXPIRY then select UPLOAD once completed
Upload Documents

How to Post Jobs

  • Once PESO has approved your registration, you will be notified in the system dashboard and the JOB POSTS function will be enabled
  • Select JOB POSTS and select ADD VACANCY / JOB POST and enter all the required information
  • Select SAVE when done
  • You can VIEW, UPDATE, MONITOR your job vacancy, or REMOVE if necessary
Post Jobs

How to Manage Applications

  • Select APPLICATION to view applicants who applied for the posted job vacancy
  • Select an action to VIEW APPLICANT, SCHEDULE AN INTERVIEW, HIRE or REJECT an application
  • Select VIEW APPLICANT icon to view the applicant's system generated CV or download the uploaded CV
  • Select SCHEDULE AN INTERVIEW icon to send a message to the applicant including interview details and platform. Select SUBMIT when done
  • Select ACCEPT / HIRE APPLICANT icon to hire the applicant. Enter the hiring date. The hired applicant profile will be reflected in the HIRED module and job vacancy counts will update
Manage Applications
