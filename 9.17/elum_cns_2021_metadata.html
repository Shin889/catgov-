Title : Existing Land Use Map-Province of Catanduanes

Edition : Edition 1

Geospatial Data Presentation Form : map

Description

Abstract : The Existing Land Use Map of the Province of Catanduanes is prepared under the project "Participatory Mapping of Existing Land Uses" for CY-2021. The Project aims to map existing land uses based on high-resolution satellite imageries, produce updated data on existing land uses through the participation of LGUS and other stakeholders and integrate existing land uses into a provincial map for monitoring land use changes and to mitigate the negative impacts of climate change. Existing Land Uses were delineated based on Google Satellite/Hybrid (2015-2022) accessed in QGIS Software, other available satellite imageries and collected maps/datasets from LGUs. The existing land use categories are in accord with the general categories indicated in the CLUP Guidebook 2013: Vol 1 (Annex 4-5. Land use Categories). Sources of information include Existing Land Use Data (LGU), Ground Validation Survey 2022 (NAMRIA & LGU), Google Satellite/Hybrid Images (2015-2022), Land Classification Map (NAMRIA), Land Cover Map 2020 (NAMRIA), National Greening Program Map (PENRO Siquijor), National Integrated Protected Areas System Map (BMB) and Topographic Map 1:50,000 Scale (NAMRIA). The Administrative Boundary reflected is based on LMB. The existing land use map was edited and finalized after the consultation meeting/workshop and field validation with LGUs and other stakeholders. For additional information, the following were delineated separately in the map: (1) closed forest within A&D as General Production Agriculture (GPA) and (2) Ricefield and Coconut as Crops (CRO). The Existing Land Use Map provides baseline information and integrated land uses and the current status of land resources of the province and will serve as input to the development and updating of Comprehensive Land Use Plans.
